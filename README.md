# Kotlin Employee Salary System

A simple Kotlin console program demonstrating inheritance, polymorphism, and collections using different types of employees.

# Features

Base Employee class with:

Protected name

Protected baseSalary

Open calculateSalary() function

FullTimeEmployee subclass with bonus pay

PartTimeEmployee subclass with hourly rate × hours worked

Uses a mapOf() to store employees and their calculated salaries

Loops through the map and prints each employee’s salary
